# 🟣 Purple Lord: Digital Odyssey

**2D-платформер о поиске своего пути в IT**

## 🎮 Описание

Игра-платформер, где вы играете за **Purple Lord** — разработчика, ищущего свой путь в мире IT-технологий. Каждый уровень представляет другую технологию (Web Dev, Data Science, Cybersecurity, GameDev, AI), а геймплей визуализирует проблему распыления и выбора своего пути.

## 🚀 Технологии

### Client
- **Phaser 3.80+** — 2D-игровой движок
- **JavaScript** — без сборщиков, чистый код
- **Tilemap** — тайловые уровни

### Server
- **Go 1.22+** — REST API
- **SQLite** — сохранение прогресса
- **net/http** — стандартная библиотека

## 🏃 Запуск

### 1. Сервер (Go)
```bash
cd server
go mod init purple-lord-server
go mod tidy
go run main.go
```
Сервер запустится на **http://localhost:3000**

### 2. Клиент (Phaser)
```bash
# Вариант 1: Простой HTTP сервер
cd client
python3 -m http.server 8080

# Вариант 2: Go сервер
cd client
go run ../server/main.go
```
Клиент запустится на **http://localhost:8080**

## 🎮 Геймплей

### Цель
Пройти уровни, представляющие разные IT-технологии, и найти свой путь.

### Механики
- **Кристаллы** — собираемые предметы, дают опыт
- **Враги** (баги, вирусы) — уничтожаются прыжком по голове или заклинанием
- **Портал** — выход с уровня, открывается после сбора всех кристаллов
- **Мана** — тратится на заклинания

### Уровни
1. **Web Development** — платформы из HTML-тегов
2. **Data Science** — облака данных, графики
3. **Cybersecurity** — цифровой код, файрволы
4. **GameDev** — пиксельная графика
5. **AI** — нейросетевые узоры

## 🏆 Особенности

- **Древо технологий** — визуализация распыления
