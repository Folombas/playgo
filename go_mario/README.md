# Go Mario — 2D Platformer

Классический 2D-платформер в стиле Super Mario Bros, написанный на Go с использованием движка Ebitengine.

## 🎮 Особенности

- 🌤️ Голубое небо с белыми облаками
- ☀️ Жёлтое солнце с лучами
- 🌱 Зелёная земля с травой
- 🎨 Пиксель-арт стиль

## 🚀 Запуск

```bash
cd go_mario
go run main.go
```

Или запустить скомпилированный бинарник:

```bash
./go_mario
```

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| ←/A | Движение влево |
| →/D | Движение вправо |
| ↑/W/SPACE | Прыжок |

## 📁 Структура проекта

```
go_mario/
├── main.go          # Основной код игры
├── go.mod           # Go модуль
├── go.sum           # Зависимости
└── go_mario         # Скомпилированный бинарник
```

## 🛠️ Технологии

- **Язык:** Go 1.x
- **Движок:** Ebitengine v2.9.9
- **Размер окна:** 800×600 пикселей

## 📝 Игровые объекты

### Небо
- Голубой фон (RGB 135, 206, 235)
- 4 белых пушистых облака
- Жёлтое солнце с лучами и свечением

### Земля
- Коричневый грунт (RGB 139, 69, 19)
- Зелёная трава (RGB 34, 139, 34)
- Детали травы (отдельные травинки)
