# 📝 CHANGELOG — Simple Snake

## [0.6.0] — 2026-03-19 (Go79)

### Добавлено

#### 🎁 Система бонусов (Power-Ups)

**6 типов бонусов:**

| Бонус | Эффект | Длительность | Цвет |
|-------|--------|--------------|------|
| ⏱️ **Slow Motion** | Замедление времени (50%) | 10 сек | Голубой |
| 🛡️ **Щит** | Неуязвимость к врагам | 8 сек | Синий |
| ✂️ **Ножницы** | Уменьшение длины змейки | Мгновенно | Зелёный |
| ❤️ **Сердце** | Дополнительная жизнь | Мгновенно | Красный |
| ⚡ **Молния** | Уничтожение всех врагов | Мгновенно | Жёлтый |
| 🍀 **Клевер** | x3 очков на 15 сек | 15 сек | Зелёный |

**Механика:**
- Спавн бонуса каждые ~10 секунд (600 тиков)
- Случайный тип бонуса при спавне
- Пульсация и свечение для привлечения внимания
- Таймер обратного отсчёта для временных эффектов

#### 💚 Система жизней
- **Начальное количество:** 1 жизнь
- **Дополнительные жизни:** от бонуса "Сердце"
- **При потере жизни:** змейка уменьшается до 3 сегментов
- **При 0 жизней:** Game Over

#### 📊 Отображение статусов
- Счётчик жизней в левом верхнем углу
- Таймеры активных эффектов в правом верхнем углу
- Формат: "XXX: Ns" (первые 3 буквы названия + секунды)

#### 🎨 Визуальные эффекты
- Частицы при сборе каждого типа бонуса
- Тряска экрана для важных бонусов
- Пульсация бонусов (amplitude 0.15)
- Свечение бонусов (мигание 5Hz)

### Изменено

#### 🎮 Геймплей
- **Щит:** защищает от врагов (уничтожает врага при столкновении)
- **Дополнительные жизни:** спасают от смерти при столкновении со стеной/хвостом/врагом
- **Множитель очков:** x3 для всех источников очков

#### 📝 Обновлён интерфейс
- Добавлено отображение жизней (Lives: N)
- Добавлено отображение активных эффектов с таймером
- Обновлены позиции UI элементов

### Технические детали

**Новые структуры:**
```go
// PowerUpType — тип бонуса
type PowerUpType int

// PowerUp — бонус
type PowerUp struct {
    Pos        Point
    Type       PowerUpType
    Active     bool
    Duration   int
    PulsePhase float64
}
```

**Новые методы в game.Game:**
```go
func (g *Game) spawnPowerUp()
func (g *Game) applyPowerUp(powerUp PowerUp) (events []GameEvent)
func (g *Game) updatePowerUps()
func (g *Game) HasShield() bool
func (g *Game) IsSlowMotion() bool
func (g *Game) GetScoreMultiplier() int
```

**Новые события:**
```go
EventPowerUpSlowMotion
EventPowerUpShield
EventPowerUpShrink
EventPowerUpExtraLife
EventPowerUpLightning
EventPowerUpMultiplier
```
