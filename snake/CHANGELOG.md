# CHANGELOG.md — Go76 (17 марта 2026)

## [0.3.0] — 2026-03-17

### Добавлено

#### 🎮 Выбор уровня сложности
- ✅ **Экран SelectDifficulty** — новый экран между меню и игрой
- ✅ **Три уровня сложности**:
  - 🟢 **Easy** — 2 жука (для новичков)
  - 🟡 **Medium** — 3 жука (стандартная сложность)
  - 🔴 **Hard** — 5 жуков (для опытных игроков)
- ✅ **Навигация по уровням** — стрелки ↑/↓ (циклический выбор)
- ✅ **Подсветка текущего уровня** — маркеры `>>` и `<<`
- ✅ **Отображение количества жуков** для каждого уровня

#### 📋 Обновлённое меню
- ✅ Подзаголовок изменён на "Go365 Go76 - Ebitengine"
- ✅ Экран выбора сложности с описанием управления

### Изменено
- 🐛 `startGame()` — динамический спавн жуков в зависимости от сложности (вместо фиксированных 10)
- 🎯 GameState — добавлено состояние `SelectDifficulty`
- 📝 Обновлены все ссылки на версию Go76

### Технические детали
- Фреймворк: Ebitengine v2.9.9
- Язык: Go 1.x
- Размер окна: 800×600 пикселей
- Размер клетки: 20×20 пикселей
- GameState enum: Menu, SelectDifficulty, Playing, Paused, GameOver
- Difficulty enum: Easy, Medium, Hard

### Управление
| Клавиша | Действие |
|---------|----------|
| ↑↓ | Выбор уровня сложности |
| ENTER/SPACE | Подтверждение уровня / Старт игры |
| ↑↓←→ | Движение змейки |
| SPACE | Выстрел стрелой |
| P | Пауза |

### Игровой процесс
1. Нажми **ENTER/SPACE** в главном меню
2. Выбери уровень сложности стрелками **↑/↓**
3. Нажми **ENTER/SPACE** для подтверждения
4. Найди **золотой ключик** 🗝️
5. Найди **пиратский сундук** 🏴‍☠️
6. Открой сундук с ключом → получи **5 стрел** 🏹
7. Стреляй в жуков (**SPACE**) 💀
8. Избегай:
   - Столкновения со стенами ☠️
   - Столкновения с хвостом ☠️
   - Жуков ☠️
   - Взрывов бомб ☠️
