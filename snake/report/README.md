## Добавлен выбор уровня сложности

### 📋 Изменения

#### 🎮 Новое состояние игры: SelectDifficulty
Добавлен экран выбора уровня сложности между меню и началом игры.

#### 📊 Уровни сложности
| Уровень | Количество жуков | Описание |
|---------|------------------|----------|
| **Easy** | 2 жука | Лёгкий уровень для новичков |
| **Medium** | 3 жука | Средний уровень сложности |
| **Hard** | 5 жуков | Сложный уровень для опытных игроков |

#### 🎯 Управление выбором сложности
- **↑/↓** — выбор уровня сложности (циклически)
- **ENTER/SPACE** — подтверждение и начало игры

### 📝 Игровой процесс
1. Запусти игру
2. Нажми **ENTER/SPACE** в главном меню
3. Выбери уровень сложности стрелками **↑/↓**
4. Нажми **ENTER/SPACE** для начала игры
5. Найди золотой ключик 🗝️
6. Открой сундук 🏴‍☠️ и получи стрелы 🏹
7. Стреляй в жуков (**SPACE**) 💀
8. Избегай столкновений!

### 🎮 Управление в игре
| Клавиша | Действие |
|---------|----------|
| ↑↓←→ | Движение змейки |
| SPACE | Выстрел стрелой |
| P | Пауза |
| ENTER | Рестарт после Game Over |
